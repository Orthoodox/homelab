# homelab

Учебный репозиторий по пути:
L1/L2 support → junior sysadmin / infra-support → junior DevOps

## Цель
Собрать фундамент по Linux, сетям, Git, автоматизации и базовым DevOps-концепциям через конспекты, практику и homelab.

## Что внутри
- `docs/glossary/` — термины
- `docs/commands/` — команды и сценарии применения
- `docs/labs/` — практические упражнения
- `docs/cheatsheets/` — краткие шпаргалки
- `scripts/` — скрипты
- `progress.md` — прогресс обучения

## Сейчас изучаю
- Ubuntu 24.04
- базовые сетевые команды Windows и Linux
- ARP / NAT / subnet / CIDR
- systemd, журналы, сервисы

## Ближайшие темы
- Linux permissions
- SSH keys
- Git basics
- network troubleshooting
