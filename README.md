# homelab

Учебный репозиторий по пути:
L1/L2 support → junior sysadmin / infra-support → junior DevOps

## Цель
Собрать практический фундамент по сетям, Linux, диагностике, Git, автоматизации и базовым DevOps-концепциям через конспекты, лабораторные работы и разбор реальных сценариев.

## Что внутри
- Networks Theory - Теория по сетям
- Linux_networks - Команды и утилиты для сетей в Linux
- Windows_Networks - Команды и утилиты для сетей в Windows

##Сейчас изучаю базовый фундамент инфраструктуры:
- сетевую диагностику Windows и Linux;
- ARP, NAT, subnet, CIDR, port, socket;
- routing table, default gateway, DNS;
- основы TCP;
- SSH-доступ к Ubuntu Server VM;
- процессы, сервисы, systemd и журналы Linux.

## Ближайшие темы
- Linux permissions
- SSH keys
- Git basics
- network troubleshooting

Ближайшая цель — подготовка к роли:

**junior sysadmin / infra-support**

DevOps рассматривается как следующий этап после закрепления:

- Linux;
- сетей;
- Git;
- автоматизации;
- Docker;
- CI/CD basics.
